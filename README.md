# Fenil Sata 3D Cinematic Portfolio

Premium cinematic portfolio built with React, Vite, Tailwind CSS, Three.js, React Three
Fiber, Drei, GSAP ScrollTrigger, Framer Motion, Lenis, and postprocessing.

## Local Setup

```bash
npm install
npm run dev
```

Open the local URL printed by Vite, usually `http://localhost:5173`.

## Production Build

```bash
npm run build
npm run preview
```

## Deployment

1. Push this folder to GitHub.
2. Import the repository in Vercel.
3. Use the default Vite settings:
   - Build command: `npm run build`
   - Output directory: `dist`
4. Deploy.
